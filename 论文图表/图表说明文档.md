### 图2-1：YOLOv8网络结构简图

**论文要求：**
- **类型**：技术示意图
- **位置**：第2章 2.2节，介绍深度学习技术部分
- **内容要求**：展示YOLOv8模型的核心架构，如Backbone、Neck、Head部分。可在YOLO官方论文或技术博客中找到标准图示，需注明引用来源。

**实际项目对应：**
- 模型文件位置：`model/yolov8s.pt`
- 实际使用：YOLOv8s版本
- 关键组件：
  - **Backbone**：CSPDarknet53结构，包含C2f模块、SPPF空间金字塔池化
  - **Neck**：FPN+PAN特征融合网络
  - **Head**：解耦检测头，分离分类和回归任务
- 输入尺寸：640×640 RGB图像
- 输出：56类民族服饰分类 + 置信度 + 边界框

**制作建议：**
- 参考YOLOv8官方架构图
- 标注数据流向（输入→Backbone→Neck→Head→输出）
- 注明各层的特征图尺寸变化

### 图4-4：智能识别服务API调用序列图

**论文要求：**
- **类型**：UML序列图
- **位置**：第4章 4.2节，智能识别模块，服务集成部分
- **内容要求**：描述一次识别请求的完整调用链：**用户前端** -> **Node.js后端** -> **Python AI服务** -> **数据库**，并返回结果的全过程，体现服务间协作。

**实际项目对应：**
- **调用链路**：
  1. 用户选择图片上传
  2. React前端预览图片
  3. POST /api/recognize → Node.js后端
  4. JWT Token验证
  5. 保存图片记录 → MongoDB
  6. POST /predict → Flask AI服务（端口5000）
  7. 图像预处理（OpenCV）
  8. YOLOv8模型推理
  9. NMS后处理
  10. 返回识别结果JSON
  11. 保存识别记录 → MongoDB
  12. 返回结果给前端
  13. 展示识别结果

- **API接口**：
  - 前端→后端：`POST /api/recognize`（端口3001）
  - 后端→AI服务：`POST /predict`（端口5000）

- **返回数据格式**：
  ```json
  {
    "ethnicity": "苗族",
    "confidence": 0.945,
    "description": "苗族服饰以银饰、刺绣为特色..."
  }
  ```

- **响应时间**：约850ms（图片上传150ms + AI推理600ms + 结果处理100ms）

**制作建议：**
- 使用UML序列图格式
- 参与者用矩形框，消息用箭头
- 区分同步调用（实线）和返回（虚线）
- 标注端口号

## 制作核心建议

### 1. 统一性与专业性
- **风格统一**：所有自制图应保持相同的配色、字体和图形风格
- **工具推荐**：
